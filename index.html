🎯 Componentes Principales

AI Agent

Actor externo inteligente que coordina y consume ambos servidores MCP para automatización completa de testing y navegación.

Coordinación Inteligente Dual Protocol Support Autonomous Decision Making

Lippia MCP Server

Servidor especializado en generación de suites de pruebas automatizadas a partir de una o más user stories.

create_feature create_test execute_test get_report

Selenium MCP Server

Servidor de automatización web para navegación autónoma, interacción con elementos y captura de datos.

start_browser navigate find_element click_element send_keys screenshot

Detalles Técnicos

Stack Tecnológico
  • Backend: Java 17+, Spring Boot, Maven
  • MCP Java SDK: Model Context Protocol
  • Automation: Lippia Framework
  • Browser: Selenium WebDriver
  • Transport: StdIO
Arquitectura
  • Microservicios: Separación de responsabilidades
  • Dual Protocol: MCP + REST APIs
  • Escalable: Arquitectura modular
  • Extensible: Fácil agregar nuevos tools
  • Resiliente: Manejo de errores robusto
