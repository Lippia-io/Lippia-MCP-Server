Getting Started

Guía completa para configurar y usar la arquitectura AI Agent + Dual MCP Servers

Sigue estos pasos para levantar los servidores MCP y comenzar a automatizar con inteligencia artificial.

Prerequisitos

Java Development Kit

Versión: Java 17 o superior

Verificar: java --version

Apache Maven

Versión: Maven 3.8+ recomendado

Verificar: mvn --version

Navegadores Web

Soportados: Chrome, Firefox

WebDrivers: Se descargan automáticamente

Instalación y Configuración

1

Clonar el Repositorio

Descarga el código fuente del proyecto desde el repositorio.

git clone https://gitlab.crowdaronline.com/lippia/products/lippia-framework/mcp-server/lippia-mcp-server.git && cd "$(basename "$_" .git)"
2

Compilar el Proyecto

Utiliza Maven para compilar y empaquetar el servidor MCP de Lippia.

mvn clean package
Resultado esperado: Se genera el archivo target/mcp-server-1.0-SNAPSHOT.jar
3

Configurar el Agente

Cursor o Claude Desktop.

Importante: Asegúrate de ingresar a la configuración de Cursor o Claude Desktop para configurar los servidores MCP que consumirá.
{
  "mcpServers": {
    "lippia": {
      "command": "java",
      "args": ["-jar", "/ruta/completa/target/mcp-server-1.0-SNAPSHOT.jar"]
    },
    "selenium": {
      "command": "npx",
      "args": [ "-y", "@angiejones/mcp-selenium" ]
    }
  }
}
4

Verificar Conectividad

Prueba que ambos servidores MCP estén funcionando correctamente.

Lippia MCP Server
- create_feature
- create_test
- execute_test
- get_report
Selenium MCP Server
- start_browser
- navigate
- find_element
- click_element
- send_keys
- screenshot

Ejemplo de Uso

Flujo Autónomo Completo

Prompt:

Hola, por favor, seguí los siguientes pasos:

1. Leer y entender la User Story propuesta

User Story: Compra exitosa de un producto en SauceDemo
Título: Como usuario logueado, quiero poder comprar un producto de forma exitosa para completar mi pedido y recibirlo.

Criterios de aceptación:

- El usuario debe poder iniciar sesión con credenciales válidas.
- El usuario debe poder agregar al menos un producto al carrito desde la tienda.
- El usuario debe poder acceder al carrito y verificar los productos seleccionados.
- El usuario debe poder proceder al checkout e ingresar su información personal (nombre, apellido, código postal).
- El usuario debe poder continuar al resumen de la orden y confirmar la compra.
- Al finalizar, el sistema debe mostrar un mensaje de confirmación de compra exitosa: "THANK YOU FOR YOUR ORDER".

Escenario de prueba (Happy Path):

- Ir a https://www.saucedemo.com.
- Iniciar sesión como standard_user / secret_sauce.
- Agregar el producto “Sauce Labs Backpack” al carrito.
- Ir al carrito y hacer clic en Checkout.

Completar los datos personales:

- First Name: John
- Last Name: Doe
- Zip Code: 12345

- Hacer clic en Continue y luego en Finish.
- Verificar el mensaje de confirmación de compra.

2. Abrí Chrome en modo normal (no headless) y navega hacia la página para conocer el flujo y mapear las acciones que realices sobre la aplicación.
3. Genera los feature files basados en la User Story apoyándote en la navegación que hiciste previamente.
4. Crea los tests en Java con Lippia MCP.
5. Ejecuta los tests en Java con Lippia MCP.

El AI Agent coordinará automáticamente ambos servidores MCP para completar todo el flujo.
